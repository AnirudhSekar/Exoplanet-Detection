## Data
This project utilizes publicly available data from both the Kepler and TESS Spacecrafts, specifically, transit method obtained data, where the Kepler Dataset is data published by NASA of confirmed exoplanets and the TESS mission data are exoplanets currently in question.

## Code

Utilizing Sklearn, Pandas, NumPy, and Matplotlib, this project reached an accuracy of almost 98% in validation due to hyperparameter optimization and the use of many features.
